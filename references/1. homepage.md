Start with the **Home Page V3 — desktop version**.

It is the best first page because it establishes the entire product’s visual identity, navigation, design system, and relationship between crowdfunding, expeditions, academy, and impact tracking.

The first mockup should include:

1. Header with logo, Donate, Explore, Academy, Impact Map, and Login.
2. Cinematic ocean hero section with three CTAs: **Donate Now**, **Join an Expedition**, and **Explore Impact**.
3. Live impact counters.
4. Three featured conservation campaigns.
5. Featured expedition destinations.
6. Interactive impact-map preview.
7. “How Terumbu.eco Works” journey.
8. Personal Impact Passport preview.
9. NGO and corporate partner logos.
10. Minimal footer.

Recommended generation order afterward:

**Home Page → Campaign Detail → User Dashboard → Trip Detail → Impact Passport → Corporate Dashboard**

The Home Page should be generated at approximately **16:9 desktop presentation format**, so it can also be shown clearly in a pitch deck.
