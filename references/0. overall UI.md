# Terumbu.eco 2.0 — Detailed UI/UX Blueprint

Design Principle:

> **Emotion → Trust → Action → Impact → Retention**

Every screen should answer:

1. Why should I care?
2. Can I trust this?
3. What should I do next?
4. What happened after I contributed?

# Overall User Journey

```text
Visitor
 ↓

Home Page
 ↓

Donation
 ↓

Impact Dashboard
 ↓

Academy
 ↓

Expedition
 ↓

Impact Passport
 ↓

Community
 ↓

Ocean Hero
```

This flow should make Terumbu.eco feel less like a donation website and more like a **conservation lifestyle platform**, similar to how Strava transformed exercise from an activity into a long-term engagement ecosystem.
